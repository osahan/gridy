Gridy

Gridy is a simple 12 Column flexbox grid system. There are three base layout modules. ‘l-container’, ‘l-grid’ and ‘l-col’ in addition to these modules there is a whole array of module modifies. These easy to use modifiers helps you restrict your changes with in a breakpoint or you can apply your style above any breakpoint. Here’s how Gridy is layed out

Getting Started

Download Gridy and include the CSS in the of your site: 


<link rel="stylesheet" href="/dist/gridy.min.css">
Alternatively, you can install Gridy with Bower or NPM.

Bower 


$ bower install gridy

npm 


$ npm install gridy

Customizing with Sass

If you’re familiar with SASS you can easily customize Gridy, its pretty easy to use your own classNames, grid gutters and even viewport breakpoints.

Import Files & Customize


$gridy-classname-prefix: "" !default;
$gridy-container-abbr: "container" !default;
$gridy-block-abbr: "grid" !default;
$gridy-column-abbr: "col" !default;
$gridy-grid-gutter: 20px !default;

$gridy-breakpoint-list: (
    xs: (0px, 320px, 100%, 20px),
    sm: (321px, 768px, 100%, 20px),
    md: (769px, 1023px, 100%, 20px),
    lg: (1024px, 1439px, 1024px, 20px),
    xl: (1440px, 100%, 1200px, 20px)
) !default;

@import "gridy/dist/app/gridy";

Understanding the base structure

Wrapper Module

.l-container className is assigned the main wrapper element, this className will bring all predefined widths to your layout based on viewport breakpoints.

This module consists of two elements, wrapper .l-container and a direct child .l-container--item and one optional modifier .lm-container--fluid, this modifier className forces .l-container to skip breakpoint width and sets its width to 100% for all breakpoints.


// Width of this container will get set based on breakpoints.
<div class="l-container">
    <div class="l-container--item">
    </div>
</div>

// Use case: Keep the wrapper container full bleed
// Width of this container will always be 100% regardless of breakpoints.
<div class="l-container lm-container--fluid">
    <div class="l-container--item">
    </div>
</div>

Grid Module

.l-grid, think of this as Bootstrap’s .row, but on steroids. It offers a set of modifiers to easily arrange columns. Every modifier has counterparts to restrict your changes to a viewport or to change the behavior above given breakpoint.


<div class="l-container">
    <div class="l-container--item l-grid">
    </div>
</div>

Grid Modifiers: Cell Direction

.lm-grid--row-reverse or use these responsive modifiers .lm-grid--*-row-reverse/ .lm-grid--above-*-row-reverse (example: .lm-grid--xs-row-reverse / .lm-grid--above-xs-row-reverse).

.lm-grid--column / .lm-grid--*-column / .lm-grid--above-*-column

.lm-grid--column-reverse / .lm-grid--*-column-reverse / .lm-grid--above-*-column-reverse

Grid Modifiers: Cell Alignment

.lm-grid--left, for responsive alignment .lm-grid--*-left or above your viewport breakpoint .lm-grid--above-*-left. (example: .lm-grid--xs-left or .lm-grid--above-xs-left)

.lm-grid--center / .lm-grid--*-center / .lm-grid--above-*-center

.lm-grid--right / .lm-grid--*-right / .lm-grid--above-*-right

.lm-grid--top / .lm-grid--*-top / .lm-grid--above-*-top

.lm-grid--middle / .lm-grid--*-middle / .lm-grid--above-*-middle

.lm-grid--bottom / .lm-grid--*-bottom / .lm-grid--above-*-bottom

Column Module

.l-col this is the base column module, you can use this to auto fill the column space.

Static Column modifier classNames

.lm-col--1 for 1/12 size column, you can go all the way up to .lm-col--12.

Responsive Columns classNames

.lm-col--*-1 to .lm-col--*-12 or .lm-col--above-*-1 to .lm-col--above-*-12 (example: replace * with your breakpoint key .lm-col--xs-1 / .lm-col--above-xs-1).

Hide on at a breakpoint or above breakpoint

.lm-col--*-hidden (example: replace * with your breakpoint key .lm-col--xs-hidden) or .lm-col--above-*-hidden (example: .lm-col--above-xs-hidden).

Column Offset .lm-cell--offset-*

.lm-col--offset-1 to .lm-col--offset-11 for static columns, for responsive columns use .lm-col--offset-*-1 or .lm-col--offset-above-*-1 (example: replace * with your breakpoint key .lm-col--offset-xs-1 or .lm-col--offset-above-xs-1).
