Flexbox based responsive grid system.

Gridy

Simple Flexbox based 12 column grid system, to create layouts through a stacking blocks of columns. Here's how Gridy is layed out


<div class="l-container">
    <div class="l-block">
        <div class="l-col">
            Content...
        </div>
    </div>

    ...
</div>
  • Stacked blocks must be placed within a .l-container or .l-container-fluid (full-width).
  • Use .l-block to vertically or horizontally stack your grid columns.
  • Use predefined classNames .l-col6 to quicking create a grid layout inside .l-block.
  • Content should be placed within .l-col* and column should be immediate child of .l-block.

Example: Grid

Start with static columns .l-col1 - .l-col12 and use viewport modifiers .lm-col-*1 - .lm-col-*12 to create a responsive grid


<div class="l-block">
    <div class="l-col lm-col-xs2  lm-col-sm3  lm-col-md4  lm-col-lg6  lm-col-xl12">
        Content...
    </div>
</div>
This column is static and the one above is responsive

Choose your own prefix



    $gridy-classname-prefix: "g";
Choose your own responsive viewport sizes 


// breakpoint: (min-screen-width, max-screen-width, max-container-width),
$gridy-breakpoint-list: (
    xs: (100%, 319px, 100%),
    sm: (320px, 767px, 100%),
    md: (768px, 1023px, 100%),
    lg: (1024px, 1439px, 1024px),
    xl: (1440px, 100%, 1200px)
);

Available classes

Container .l-container

.l-container is the main layout className with predefined width based on viewport breakpoints, you need atleast one .l-block element as a direct child of this parent element. 


<div class="l-container">
    <div class="l-block">
        ...
    </div>
</div>

Full bleed container with .lm-container--fluid modifier

In case you want a full bleed container, you can use .lm-container--fluid modifier class with .l-container 


<div class="l-container lm-container--fluid">
    <div class="l-block">
        ...
    </div>
</div>
